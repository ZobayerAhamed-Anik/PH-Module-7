Java vs JavaScript:
Ans:In Java:
    -Created by Sun Microsystems(1995)
    -Used for Android apps, enterprise software
    -Must be compiled before running
    -Strictly typed - every variable has a fixed type
While in JavaScript
    -Created by Netscape(also 1995)
    -Used for websites, web apps, servers
    -Runs directly - no compilation step
    -Loosely typed - variables can hold any value

What JavaScript Can Do?
Ans:
    1. User Interactions:
    Responds to clicks, keyboard input, and mouse events

    2. Dynamic Content:
